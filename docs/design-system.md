# 户外公积金管理系统 - 设计系统

## 设计理念

户外活动 = 自然、活力、信任。以山岳绿为主色调，搭配暖日橙强调色，
传递户外运动的自然感与资金管理的可信赖感。移动端优先，卡片式布局，
大字号余额展示，操作路径简短。

---

## 色彩调色板

### 主要颜色
- 山岳绿 `#10B981` — 主操作按钮、Tab 激活态、余额正数
- 深林绿 `#059669` — 按钮 hover/pressed
- 墨绿 `#064E3B` — 顶栏背景、深色文字

### 次要颜色
- 薄荷浅绿 `#D1FAE5` — 标签背景、成功提示底色
- 雾灰绿 `#ECFDF5` — 页面背景

### 强调色
- 暖日橙 `#F59E0B` — 管理员标签、警告、充值按钮
- 深橙 `#D97706` — 橙色 hover

### 功能性颜色
- 成功绿 `#10B981`
- 错误红 `#EF4444`
- 警告橙 `#F59E0B`
- 信息蓝 `#3B82F6`
- 中性灰 `#6B7280`

### 背景颜色
- 页面底 `#F0FDF4` (green-50)
- 卡片白 `#FFFFFF`
- 分割线 `#E5E7EB`

## 排版

- 主字体: system-ui, -apple-system, sans-serif
- 余额大字: 32px / Bold
- 页面标题: 18px / Semibold
- 正文: 14px / Regular
- 辅助文字: 12px / Regular, color #9CA3AF

## 组件约定 (Naive UI Theme Override)

### 按钮
- 主按钮: bg #10B981, 圆角 12px, 高 44px
- 危险按钮: bg #EF4444
- 充值按钮: bg #F59E0B

### 卡片
- 圆角 16px, shadow-sm, padding 16px
- 会员卡片: 左侧头像圆圈 + 右侧信息

### 输入框
- 圆角 10px, 高 44px, border #D1D5DB
- focus border #10B981

### 底部 Tab
- 高 56px, 5 等分
- 激活态: 山岳绿图标 + 文字
- 非激活: #9CA3AF
